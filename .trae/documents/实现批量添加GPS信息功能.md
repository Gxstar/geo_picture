# 实现批量添加GPS信息功能

## 需求分析
1. 增加批量选择图片的功能
2. 避免加载所有缩略图，只加载文件名列表
3. 点击图片时才显示对应的预览
4. 支持批量添加GPS信息到所有选中的图片

## 实现方案

### 1. 前端HTML修改
- 增加批量选择图片的按钮
- 增加图片列表区域，显示选中的文件名
- 保持现有的图片预览区域，只在点击列表项时显示对应图片
- 增加批量处理按钮

### 2. 前端JavaScript修改
- 增加全局变量存储选中的文件列表
- 增加selectMultipleFiles函数，调用后端API选择多个文件
- 增加renderFileList函数，渲染文件列表
- 增加loadSelectedImage函数，点击列表项时加载对应图片
- 增加addGeoTagToAll函数，批量添加GPS信息到所有选中图片

### 3. 后端Python修改
- 在Api类中增加select_multiple_files方法，支持选择多个文件
- 在Api类中增加process_multiple_images方法，批量处理图片

### 4. 性能优化
- 只在点击图片时才加载对应图片的缩略图
- 批量处理时使用异步方式，避免UI卡死
- 显示处理进度，让用户了解处理状态

## 预期效果
- 用户可以选择多个图片文件，只显示文件名列表
- 点击列表中的文件名，才会加载并显示对应的图片预览
- 可以为所有选中的图片批量添加相同的GPS信息
- 批量处理时显示进度，避免UI卡死

## 实现步骤
1. 修改后端API，增加批量选择和处理图片的方法
2. 修改前端HTML，增加批量选择按钮和文件列表区域
3. 修改前端JavaScript，实现批量选择、文件列表渲染和批量处理功能
4. 测试功能，确保性能和用户体验良好
